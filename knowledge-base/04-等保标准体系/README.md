## 使用方式

- **直接查阅：** 标准PDF名即为标准编号+名称，可直接用搜索定位
- **文档引用：** 在加固方案/测评报告中引用时，标注标准号+版本
- **交叉比对：** 等保2.0扩展要求（云计算/移动互联/物联网/工控）可供新场景测评参考
