# 03-安全运维

> 安全运维与值守服务

## 服务概述

安全运维是安服团队的持续性业务，包括日常安全监控、安全设备运维、安全加固、漏洞管理、安全巡检等。云贵区域客户对安全运维的需求以政务云、大数据中心、运营商为主。

## 运维内容

### 1. 日常监控
- 安全设备运行状态监控
- 安全事件实时告警处理
- 流量异常监测

### 2. 安全运维
- 防火墙策略维护与优化
- IPS/WAF特征库升级与规则调优
- VPN/SSL证书管理
- 日志审计系统维护

### 3. 漏洞管理
- 定期漏洞扫描
- 漏洞验证与风险评级
- 跟踪漏洞修复进度

### 4. 安全巡检
- 设备健康检查（CPU/内存/磁盘/会话数）
- 日志完整性检查
- 策略合规性检查
- 巡检报告编制

## 实战Tips

- **巡检优先级**：CPU/内存/磁盘→日志→策略，按此顺序排查问题最快
- **变更管理**：防火墙策略变更必须有申请→审批→执行→确认流程
- **文档化**：每一次运维操作都要记录，这是等保测评的审计要求
- **云贵区域特点**：客户运维人员流动大，设备交接时重点检查配置备份是否完整

## 关键指标

- 安全事件响应时间：< 30分钟（告警类）
- 漏洞修复周期：高危7天/中危30天/低危90天
- 设备可用率：≥ 99.9%
- 巡检频率：月度/季度（按合同约定）
