# 天融信安全服务产品线整体介绍

天融信安全服务团队由"业务中心+能力中心"组成。基于天融信在安全攻防、安全工程方面的研究、实践和应用，以为业务保驾护航为目标，为客户提供信息系统全生命周期中多元化、多层级的安全服务。

**天融信安全服务理念：**以专业精神、专业技能、专业品质，向客户提供高质量的专业安全技术服务，并保持持续创新以满足客户日益变化的安全需求。

**天融信安全服务愿景：**术业专攻，成就所托。

业务中心：全国共建立北京、郑州、上海、西安、成都、华南、安全集成、安全托管服务（MSS）等业务中心，1000+名专业安全服务人员，结合天融信覆盖全国的90余个分支机构，为客户提供优质的安全服务。

能力中心：阿尔法实验室、赤霄实验室、听风者实验室、网空安全对抗中心和安全威胁研究中心，分别负责网络安全领域的攻击技术的研究和利用、APT追踪、威胁狩猎以及攻击技术的集成化输出、国内外网络安全前沿技术研究与网络安全威胁防护体系构建。为业务中心提供充分的理论与实践基础，是提升业务中心能力的重要能力机构。

从公司成立至今30余年，天融信安全服务团队参与了大量政府、电信运营商、能源、金融及企事业单位的安全服务行业项目实施，积累了丰富的项目实施经验，并在历年国家组织的攻防演练行动中屡创佳绩，连续多年获得"最佳攻击团队"荣誉；参与全国各省级别攻防演练，成绩始终排名前三；组织参与全国地市级别攻防演练数百场，各地市均对天融信技术能力表示高度认可。

截至目前，天融信已形成覆盖安全服务各领域的完整资质矩阵。不仅是首批入选并连续十一届获评CNCERT最高级网络安全应急服务支撑单位，也是国家重大网络安全保卫技术支持单位、国家信息安全漏洞库CNNVD技术支撑单位，首批入选的公安部认证的信息安全等级保护安全建设服务机构；具有中国信息安全测评中心及中国网络安全审查技术与认证中心颁发的多个最高级别安全服务认证资质，安全服务能力常年保持业内领先地位。

专利与国家行业标准方面，具有超过100项安全专利，参与110项国家级和行业级标准制定，包含网络安全、云安全、大数据安全、工控安全等各个领域，也是中国网络安全产业联盟、网络安全与信息化产业联盟的常务理事单位，CSA、云计算产业联盟的会员，工控领域是工信安全产业联盟会员。

# 天融信安全服务检测、分析能力

1、安全运营中心成立

天融信成立覆盖全国范围的安全运营中心，专职为客户提供安全服务业务，提升检测、分析、处置能力的同时，快速解决客户安全问题。

2、能力建设和知识库积累

天融信安服为了提升自身的安全能力建设，更好的服务客户，积累的安全知识库包括：漏洞库、规则库、分析研判库，安服人员通过知识库快速检测和定位漏洞，针对高危事件和安全设备检测能力不足的情况，提供深层次安全检测和安全预警。

3、自动化安全检测和分析能力

天融信安服深入拓展自动化安全检测和分析能力，建立安全规则模型和行为模型，实现自动化安全检测，快速发现安全事件，实现自动化研判，减少人工投入；同时针对安全设备无预警、绕过、内网失陷、主机失陷、违规外联等情况，提供安全检测和预警，弥补安全设备的检测不足和准确性问题。

4、模型和规则运营能力

基于客户现有的安全能力，天融信安服人员结合客户业务环境，深入优化安全设备检测规则，建立一套适合客户系统的安全模型，提升客户的安全检测和安全运营能力。

# 天融信安全服务处置能力

1、自动化编排运营-SOAR能力建设

为了提升和建设客户安全处置能力，天融信安全服务运营中心，结合客户安全环境和安全设备，设计'安全剧本'，实现基础的安全处置和自动化，快速处置安全事件，解决安全事件响应严重依赖人工操作的问题。

2、快速应急和自动化

天融信安全服务结合多年的应急和处置经验，研发了一套快速应急响应平台，自动分析安全事件结果；辅助安全服务工程师快速定位安全事件，缩短安全事件应急处置时间，减少客户损失。
