# 05-重保

> 重大活动安全保障

## 服务概述

重保（重大活动网络安全保障）是云贵区域安服团队的季节性重点业务。覆盖两会/党代会/进博会/亚运会等国家级活动，以及省内重大会议、重要时期。重保周期通常为7-30天，包含备战、临战、实战、总结四个阶段。

## 保障阶段

### 第一阶段：备战（T-30 至 T-7）
- **资产梳理**：摸清所有在网资产，更新资产清单
- **漏洞排查**：全网扫描，高危漏洞清零
- **策略加固**：防火墙策略最小化收敛
- **应急预案**：更新应急处置预案，组织演练
- **值班排班**：确定7×24值班人员安排

### 第二阶段：临战（T-7 至 T-1）
- **全量巡检**：所有安全设备健康检查
- **日志检查**：日志服务器空间、日志完整性
- **联动测试**：防火墙-IPS-WAF联动测试
- **容灾检查**：备机/冷备设备就绪情况

### 第三阶段：实战（T日 至 T+N）
- **7×24值守**：三班倒监控+分析+处置
- **1-5-10原则**：1分钟发现→5分钟研判→10分钟处置
- **每小时汇报**：安全态势零报告
- **0点总结**：每日安全态势总结

### 第四阶段：总结
- 重保工作总结报告
- 攻击事件统计分析
- 安全策略优化建议
- 资产清单更新归档

## 实战Tips

- **云贵区域关注**：省级政务云是重保重点单位，需提前对接协调
- **值班效率**：建立快速研判SOP，常见告警30秒内判断是否误报
- **应急处置提前准备**：封禁API脚本、应急阻断模板、话术模板提前写好
- **报告模板**：日报/总结报告模板提前落地，避免临场写报告

## 关键指标

- 安全事件发现时效：≤ 1分钟
- 安全事件研判时效：≤ 5分钟
- 安全事件处置时效：≤ 10分钟
- 误报率控制：≤ 30%
