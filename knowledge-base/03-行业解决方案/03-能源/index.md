# 03-能源行业安全解决方案

## 1. 行业概述

能源行业是国家关键基础设施的核心组成部分，涵盖电力、石油、石化、天然气等领域。主要业务场景包括：

- **电力**：电力调度（SCADA/EMS/DMS）、发电（火电/水电/风电/光伏）、输变电、配电、售电
- **石油石化**：勘探开发、油气管道SCADA、炼化生产控制、油库管理、加油（气）站
- **新能源**：风电/光伏场站集中监控、储能系统管理、虚拟电厂

能源行业安全的核心特征：**生产控制安全优先于信息安全**（Safety First），工控系统（ICS/SCADA）实时性要求高，**IT与OT深度融合**带来新型风险，且一旦遭受攻击后果严重（断电停产、设备损坏、人身安全威胁）。

## 2. 安全风险分析

### 2.1 工控系统（OT）安全风险

| 风险类别 | 风险描述 | 影响等级 |
|---------|---------|---------|
| 协议漏洞 | Modbus/IEC 104/DNP3等工控协议缺乏认证加密 | 严重 |
| PLC/RTU攻击 | 控制器被篡改逻辑，导致设备异常运行 | 严重 |
| 上位机感染 | 操作员站/工程师站被植入恶意软件 | 严重 |
| 固件篡改 | 智能设备固件被植入后门 | 严重 |
| 无线攻击 | 风电/光伏场站、管道的无线传感器网络攻击 | 高风险 |

### 2.2 电力行业特有风险

| 风险类别 | 风险描述 | 影响等级 |
|---------|---------|---------|
| 调度系统攻击 | 电网调度指令被篡改，引发大面积停电 | 严重 |
| 负荷控制劫持 | 攻击者控制负荷管理系统，恶意甩负荷 | 严重 |
| 发电控制系统 | DCS/PLC被攻击导致发电机组失控 | 严重 |
| 新能源监控 | 风场/光伏集中监控系统被攻击 | 高风险 |
| 智能电表 | 电表安全芯片被破解，计量篡改 | 中风险 |

### 2.3 石油石化特有风险

| 风险类别 | 风险描述 | 影响等级 |
|---------|---------|---------|
| 管道SCADA | 油气管道压力/流量控制被篡改，引发泄漏爆炸 | 严重 |
| 炼化DCS | 炼化装置过程控制异常，导致安全事故 | 严重 |
| 油库管理 | 油库液位/阀门控制被篡改 | 高风险 |
| 加油加气站 | POS系统被攻击、支付数据泄露 | 中风险 |

## 3. 安全解决方案

### 3.1 总体架构

能源行业采用 **"安全分区、网络专用、横向隔离、纵向认证"** 的工控安全总体策略：

```
┌──────────────────────────────────────────────────────────────────┐
│                      能源安全运营中心                              │
│      （SOC + 工控安全态势感知平台 + ICS-CERT应急响应）               │
├─────────────┬─────────────┬──────────────┬──────────────────────┤
│ 安全I区      │ 安全II区     │ 管理信息大区    │ 对外服务区           │
│ （实时控制）  │ （非实时控制）│              │                      │
├─────────────┼─────────────┼──────────────┼──────────────────────┤
│ ·电力调度    │ ·电能量采集  │ ·MIS系统     │ ·营销系统            │
│ ·SCADA/EMS  │ ·水调自动化  │ ·OA/ERP      │ ·网上营业厅          │
│ ·DCS        │ ·故障录波   │ ·设备管理     │ ·客户服务            │
│ ·RTU/PLC    │ ·保信子站   │ ·人资/财务    │ ·APP/微信服务        │
├─────────────┼─────────────┼──────────────┼──────────────────────┤
│ 正反向隔离    │              │ 防火墙/IDS    │ WAF/API网关         │
│ 装置          │              │              │                      │
└─────────────┴─────────────┴──────────────┴──────────────────────┘
```

### 3.2 工控安全隔离体系

#### 3.2.1 安全分区与隔离

| 隔离层级 | 隔离技术 | 说明 |
|---------|---------|------|
| 控制区↔非控制区 | 逻辑隔离（防火墙+访问控制） | I区与II区之间 |
| 控制区↔管理区 | 物理隔离（正反向隔离装置） | 专用单向隔离装置 |
| 管理区↔互联网 | 逻辑隔离（防火墙+WAF+IDS） | 管理信息大区边界 |

#### 3.2.2 纵向认证加密

- 调度主站↔厂站之间的通信采用**基于国密算法的身份认证与加密**
- 支持IEC 104/IEC 61850/Modbus TCP等协议的国密改造
- 纵向加密认证装置部署在各级调度和厂站侧
- 数字证书管理体系，实现设备级身份认证

### 3.3 OT安全防护方案

#### 3.3.1 工控主机安全

- **白名单机制**：仅允许已授权的进程/脚本/应用运行
- **USB管控**：仅允许授权的加密U盘，禁用普通USB存储
- **补丁管理**：工控系统补丁需经过严格兼容性测试后分批推送
- **主机加固**：禁用不必要的端口和服务，关闭高危端口
- **防病毒**：工控专用防病毒（轻量级、低资源占用）

#### 3.3.2 工控网络异常检测

```
                    ┌─────────────────────┐
                    │ 工控安全态势感知平台   │
                    └──────┬──────────────┘
                           │
    ┌──────────────────────┼──────────────────────┐
    │                      │                       │
 ┌──┴──┐             ┌─────┴─────┐          ┌─────┴─────┐
 │流量探针│             │日志探针     │          │威胁情报     │
 │(网络层)│             │(主机层)     │          │(全局)      │
 └──────┘             └───────────┘          └───────────┘
```

- **深度协议解析**：支持Modbus、IEC 104/101、DNP3、PROFINET、OPC UA等20+工控协议
- **基线学习**：自动学习工控网络流量基线，识别异常流量和非法操作
- **指令级检测**：检测异常的PLC指令（如Stop、Reset、Download）
- **资产画像**：自动发现工控设备、通信关系、应用依赖

#### 3.3.3 工控漏洞与补丁管理

- **工控资产漏洞扫描**：专用工控漏洞扫描工具，不影响生产运行
- **漏洞生命周期管理**：发现→评估→防护→修复→验证
- **虚拟补丁**：对无法修复的工控系统实施虚拟补丁（IPS规则缓解）
- **供应链安全**：工控设备固件安全检测、软件物料清单（SBOM）

### 3.4 电力调度安全增强方案

#### 3.4.1 调度数据网安全

- **MPLS-VPN**：安全I区/II区逻辑隔离通道
- **国密纵向加密**：主站↔厂站双向身份认证 + 数据加密
- **异常流量检测**：调度数据网流量分析，检测异常通信模式
- **时钟同步安全**：NTP/北斗时钟源认证，防止时间攻击

#### 3.4.2 发电控制系统安全

- **DCS安全加固**：控制器访问控制、组态修改审批、操作权限分级
- **DCS边界防护**：DCS与其他系统间的工业防火墙/单向隔离
- **工程师站安全**：双因子认证、操作审计、U盘管控
- **AGC/AVC安全**：自动发电控制/自动电压控制指令验签

### 3.5 石油石化安全增强方案

#### 3.5.1 管道SCADA安全

- 管道SCADA系统与办公网络物理隔离
- 远程终端单元（RTU）通信使用国密加密
- 管道压力/流量控制指令签名验证
- 泄漏检测系统防篡改

#### 3.5.2 炼化DCS安全

- DCS系统网络分段与控制网络隔离
- 操作员站/工程师站白名单管控
- 安全仪表系统（SIS）独立于DCS
- 紧急停车系统（ESD）不受网络安全事件影响

### 3.6 工控安全运营中心

- **统一监控**：融合IT/OT、物理安全的统一安全监控
- **工控安全态势感知**：可视化展示全网工控安全状态
- **工控威胁情报**：能源行业专享威胁情报（IoC、TTP）
- **应急响应**：工控安全事件专项响应预案与演练（每年2次以上）
- **工控SOC**：7×24工控安全专家值守

## 4. 合规要求

| 法规标准 | 核心要求 | 对应方案 |
|---------|---------|---------|
| 等保2.0（工控扩展要求） | 工控系统安全保护 | 安全分区、隔离、认证 |
| 《电力监控系统安全防护规定》（发改委14号令） | 安全分区、网络专用、横向隔离、纵向认证 | 全覆盖设计 |
| 《电力行业网络安全管理办法》 | 电力企业网络安全责任 | 组织体系建设 |
| 《石油石化系统安全防护要求》 | 石油石化工控系统防护 | 行业定制方案 |
| 《关键信息基础设施安全保护条例》 | CII认定/保护/检测 | 专项保障体系 |
| 工信部《工业控制系统信息安全行动计划》 | 工控安全能力提升 | 持续改进机制 |

## 5. 典型客户案例

### 案例一：某省级电网调度安全防护体系

为某省级电网公司建设电力调度安全防护体系：
- 覆盖调度主站+300+变电站/电厂的整体安全防护
- 正反向隔离装置部署率100%，国密纵向加密全覆盖
- 工控安全态势感知平台日处理10亿+条工控协议流量
- 成功发现并阻断3起针对SCADA系统的APT攻击
- 通过等保2.0（三级）和《电力监控系统安全防护规定》合规检查

### 案例二：某大型石化集团工控安全建设

为某石化集团下属8家炼化企业、20+油库实施工控安全：
- DCS/PLC网络全面安全加固，白名单部署3000+台工控主机
- 管道SCADA系统物理隔离与加密认证改造
- 连续2年零安全事件，生产安全故障率下降60%
- 通过工信部工业互联网安全标杆企业认定
