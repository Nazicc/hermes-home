# 烟草行业安全解决方案

**编号：07-烟草 | 适用行业：行业专网/工控/营销系统**

---

## 一、行业背景与安全挑战

### 1.1 行业概述

中国烟草行业实行"统一领导、垂直管理、专卖专营"的管理体制，拥有覆盖全国的行业专网，以及从烟叶种植、复烤加工、卷烟生产到物流配送、市场营销的完整产业链。行业信息化系统主要包括：

- **行业专网**：连接国家局、省级局（公司）、地市级局、县级局的四级专网
- **工业控制系统**：制丝线、卷接包生产线、物流自动化线等MES/SCADA系统
- **营销管理系统**：卷烟销售、订货、物流配送、零售终端管理
- **办公管理系统**：OA、财务、人事、专卖许可证管理等

### 1.2 核心安全挑战

| 风险领域 | 具体挑战 |
|----------|----------|
| **行业专网安全** | 四级专网横向联通、纵向贯通，边界防护不足易导致攻击横向移动 |
| **工控系统安全** | 卷烟生产MES/SCADA系统老旧，大量使用Windows XP/7，补丁更新困难；生产线停机损失巨大（单条产线日产值千万级） |
| **营销系统安全** | 全国数百万零售户通过订货平台交易，Web应用攻击和大规模DDoS风险高 |
| **数据安全** | 行业经营数据、零售户信息、消费者会员数据涉及商业机密和个人信息保护 |
| **供应链安全** | 烟机设备大量进口，PLC/DCS存在供应链后门风险 |

### 1.3 典型攻击场景

- 攻击者通过钓鱼邮件渗透省级公司办公网，横向移动至生产网，篡改配方参数造成批量次品
- 营销订货平台遭CC攻击，导致全省零售户无法下单订货
- 行业专网DNS劫持，将物流调度信息重定向
- 勒索软件加密卷烟生产MES服务器，导致整条产线停产

## 三、安全解决方案总体架构

### 3.1 "三横两纵"安全架构

```
┌──────────────────────────────────────────────────────────┐

│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐        │
│  │安全态 │ │威胁情 │ │合规管 │ │应急响 │ │态势大 │        │
│  │势感知 │ │报平台 │ │理平台 │ │应平台 │ │屏    │        │
│  └──────┘ └──────┘ └──────┘ └──────┘ └──────┘        │
├──────────────────────────────────────────────────────────┤
│  第一横：行业专网安全    │  第二横：工控系统安全    │  第三横：营销系统安全  │
├──────────────────────────────────────────────────────────┤
│ ┌────────────────────┐ ┌──────────────────┐ ┌──────────┐ │
│ │ 四级专网边界防护     │ │ 生产网纵深防御     │ │ 营销平台   │ │
│ │ • 省-地市边界防火墙  │ │ • 制丝线安全隔离   │ │ Web防护  │ │

│ └────────────────────┘ └──────────────────┘ └──────────┘ │
├──────────────────────────────────────────────────────────┤
│  纵向安全运营       │       纵向数据安全                     │
│ ┌────────────────────┐ ┌────────────────────────────────┐ │
│ │ 国家局→省局→市局    │ │ 经营数据加密、脱敏、水印             │ │
│ │ 三级SOC联动        │ │ 零售户信息保护、消费者隐私保护       │ │
│ │ 统一安全策略下发    │ │ 行业数据跨境合规（进口烟叶数据）     │ │
│ └────────────────────┘ └────────────────────────────────┘ │
└──────────────────────────────────────────────────────────┘
```

### 3.2 设计原则

- **统一规划、分级部署**：国家局统一规划安全架构，省局/市局分级部署
- **业务优先、安全融入**：安全措施不影烟草生产连续性和营销时效性
- **自主可控**：优先采用国密算法和国产安全设备
- **数据驱动**：以数据安全治理为核心，实现全生命周期保护

## 四、分场景解决方案

### 4.1 行业专网安全

#### 4.1.1 四级专网拓扑

```
国家局 ── (骨干网) ── 省级公司 ── (城域网) ── 地市级公司 ── (接入网) ── 县级公司
  │                      │                      │                      │
┌──────┐              ┌──────┐              ┌──────┐              ┌──────┐
│数据中心│              │省公司 │              │市公司 │              │县公司 │
│安全域 │              │安全域│              │安全域│              │安全域│
└──────┘              └──────┘              └──────┘              └──────┘
```

#### 4.1.2 专网安全防护措施

| 防护措施 | 说明 |
|----------|------|
| **边界防火墙** | 各级单位出口部署防火墙，实施基于应用的精细化访问控制 |
| **IPS入侵防御** | 骨干网节点部署IPS，检测和阻断已知攻击 |
| **VPN加密传输** | 省-地市-县三级链路采用国密IPSec VPN加密 |
| **流量可视化** | 部署NPMD网络性能监测，实时掌握专网运行状态 |
| **DNS安全** | 行业DNS递归解析安全防护，防止DNS劫持和缓存投毒 |
| **邮件安全** | 行业邮件系统部署反垃圾、反钓鱼、邮件DLP |
| **统一认证** | 行业PKI/CA体系，数字证书统一身份认证 |

#### 4.1.3 远程接入安全

- 移动办公、异地出差通过SSL VPN接入行业专网
- 接入终端强制安装EDR，通过终端安全基线检查后方可接入
- 远程操作全程录像审计，敏感操作需二次审批

### 4.2 工控系统安全

#### 4.2.1 烟草生产网络架构

```
┌─────────────────────────────────────────────┐
│         生产管理层 (MES/ERP)                   │
│    生产计划、质量追溯、设备管理、仓储管理          │
└──────────────────┬──────────────────────────┘
                   │ 工业防火墙/网闸
┌──────────────────▼──────────────────────────┐
│         过程控制层 (SCADA/DCS)                 │
│    制丝集控、卷接包监控、动力能源监控             │
└──────┬───────────────────┬───────────────────┘
       │                   │
┌──────▼──────┐    ┌──────▼──────┐
│ 制丝线PLC    │    │ 卷接包PLC    │
│ (SIEMENS)   │    │ (Rockwell)  │
│ 切丝/烘丝/加香│    │ 卷接机组/包装 │
└──────┬──────┘    └──────┬──────┘
       │                   │
┌──────▼──────┐    ┌──────▼──────┐
│ 现场仪表      │    │ 输送/码垛    │
│ 传感器/执行器 │    │ AGV/机器人   │
└─────────────┘    └─────────────┘
```

#### 4.2.2 工控安全措施

| 防护方向 | 具体措施 |
|----------|----------|
| **网络隔离** | MES层与SCADA层之间部署工业防火墙，仅允许特定协议（OPC UA/Modbus TCP）通信 |
| **主机加固** | 操作员站、工程师站安装白名单软件，禁止未授权程序运行；禁用USB口 |
| **协议审计** | 部署工控网络审计系统，对OPC、Modbus、Profinet等协议进行深度解析和行为审计 |
| **漏洞管理** | 生产系统上线前进行工控漏洞扫描，建立工控漏洞库 |
| **补丁管理** | 建立工控系统补丁测试环境，先测试后上线；无法打补丁的系统用虚拟补丁/IDS补偿 |
| **变更管理** | PLC程序变更、配方变更需走审批流程，变更前后自动比对 |
| **备份恢复** | PLC程序、HMI工程、SCADA数据库定时备份，支持快速恢复 |

#### 4.2.3 制丝线重点防护

制丝生产线是卷烟生产核心工序，包括切片、切丝、烘丝、加香加料等环节：
- 烟草配方参数（温度、湿度、流量、比例）是核心商业机密，需加密存储
- 非法篡改配方参数可能导致整批烟叶报废
- 部署配方管理系统防篡改模块，任何参数修改需双人授权
- 关键参数写入PLC前进行合法性校验（范围检查、合理性检查）

#### 4.2.4 物流自动化安全

- AGV/堆垛机/穿梭车无线通信（WiFi/5G）加密认证
- 物流调度系统（WMS/WCS）与PLC之间工控防火墙隔离
- 高架库RFID读写器安全接入，防止标签克隆
- 成品烟自动装车系统防篡改，确保出库数据准确

### 4.3 营销系统安全

#### 4.3.1 营销系统架构

```
┌─────────────────────────────────────────────┐
│          营销综合管理平台                        │
│  货源分配 ↓ 订单处理 ↓ 物流配送 ↓ 客户服务       │
└──────────┬──────────────────┬────────────────┘
           │                    │
┌──────────▼──────┐  ┌─────────▼─────────┐
│  订货平台        │  │  零售户终端        │
│  Web门户/APP    │  │  新商盟/订货APP    │
│  API接口        │  │  终端扫码系统      │
└──────────┬──────┘  └─────────┬─────────┘
           │                    │
┌──────────▼──────────────────▼─────────┐
│          银行/支付接口网关                 │
│  银联、微信、支付宝支付                    │
└─────────────────────────────────────────┘
```

#### 4.3.2 营销系统安全措施

| 安全领域 | 防护措施 |
|----------|----------|
| **Web安全(WAF)** | 部署Web应用防火墙，防御SQL注入、XSS、CSRF、文件上传漏洞等OWASP Top 10攻击 |
| **API安全** | 全量API网关统一管控，实施认证、限流、参数校验、敏感数据脱敏 |
| **抗DDoS** | 全国订货高峰日（如每月25日），流量可达正常数百倍，需部署云清洗+本地清洗联动 |
| **爬虫防护** | 防止恶意爬虫批量抓取货源信息、零售户数据 |
| **业务安全** | 防止囤货刷单、恶意退货、虚假注册等业务欺诈行为 |
| **终端安全** | 零售户订货APP需加固（加壳、反调试），防止逆向分析 |
| **支付安全** | PCI-DSS合规，支付数据不落地，Token化处理 |

#### 4.3.3 亿级用户数据保护

全国数百万零售户的敏感信息包括：
- 身份信息：姓名、身份证号、烟草专卖许可证号
- 联系方式：手机号、地址
- 经营信息：订货记录、销售额、利润数据
- 银行账户：结算账户信息

**保护措施：**
- 数据库加密存储（TDE），密钥与数据分离管理
- 应用层脱敏：客服系统查看零售户信息时自动脱敏中间4位手机号
- 数据访问审计：每次查询记录操作人、时间、查询内容
- 大数据分析数据脱敏后方可出库

## 五、数据安全专项

### 5.1 数据分类分级

| 安全等级 | 数据类别 | 保护要求 |
|----------|----------|----------|
| 机密 | 国家局决策数据、全国产销存数据 | 加密 + 访问审批 + 审计 |
| 敏感 | 省级经营数据、零售户详细信息、配方参数 | 加密 + 权限控制 + 脱敏 |
| 内部 | 日常经营报表、OA文档、公文 | 权限控制 + 传输加密 |
| 公开 | 政策法规、行业新闻、公开招标 | 完整性校验 |

### 5.2 数据安全防护

- **DLP数据防泄漏**：邮件、文件传输、打印等通道防止敏感数据外泄
- **数据库安全**：数据库防火墙、数据库审计、数据库加密
- **水印溯源**：行业报表嵌入显性/隐形水印，泄露可追溯
- **数据销毁**：废弃设备数据擦除，符合国标数据销毁规范

## 六、安全运营中心（SOC）

### 6.1 三级SOC联动体系

```
国家局SOC（一级）
├── 宏观态势、威胁情报、全局策略
├── 跨省事件研判与应急协调
│
├── 省局SOC（二级）
│   ├── 本省安全态势监控
│   ├── 工控安全事件处置
│   └── 营销平台安全运营
│
│   └── 市局SOC（三级）
│       ├── 属地资产监控
│       └── 基础安全事件处理
```

### 6.2 SOC核心能力

- **资产发现与管理**：自动发现全省IT/OT资产
- **日志集中管理**：防火墙、IPS、工控审计、EDR、WAF日志统一接入
- **告警关联分析**：基于MITRE ATT&CK的攻击链关联分析
- **SOAR自动化**：常见告警（如暴力破解、恶意软件）自动封禁
- **工控专有检测**：针对烟草行业业务模型的异常检测（如配方异常修改）

## 八、建设路径规划

### 阶段一：基础合规（0-6个月）
- 完成等保定级、备案、测评工作
- 行业专网边界防火墙部署，实现安全域隔离
- 部署终端防病毒和EDR
- 建立安全管理组织和制度体系

### 阶段二：工控安全防护（6-12个月）
- 生产网工控安全隔离（工业防火墙、网闸）
- 操作员站白名单加固
- 工控协议深度审计部署
- MES/SCADA系统漏洞评估和加固

### 阶段三：全面安全运营（12-24个月）
- 省-市两级SOC建设
- 营销平台全栈安全防护（WAF+API+抗D+爬虫）
- 数据安全治理（分类分级、DLP、脱敏）
- 三级SOC联动运营

## 九、典型案例

### 案例一：某省烟草工业公司工控安全建设
- **规模**：3个卷烟厂、12条制丝线、24台卷接包机组
- **方案**：生产网MES-SCADA隔离、操作站白名单、工控网络审计、SOC集中管理
- **效果**：发现并阻断2起境外IP通过邮件横向渗透至生产网的事件，生产系统连续运行无中断

### 案例二：某省烟草公司专网+营销安全整体方案
- **规模**：覆盖全省15个地市、100+县区、15万零售户
- **方案**：专网国密VPN + WAF + API网关 + 抗DDoS + 零售户数据加密
- **效果**：订货高峰系统可用性99.99%，零售户信息安全合规通过检查

## 十、总结

烟草行业作为国家专卖专营的关键行业，其网络安全建设需要兼顾行业专网、工控生产系统和营销平台三个维度的特殊安全需求。本方案基于"统一规划、分级部署、业务优先"的原则，提供覆盖IT/OT/数据全场景的安全防护体系，助力烟草行业在数字化转型过程中实现"安全合规、生产稳定、数据可靠"的目标。

> **关键建议**：以等保2.0合规为基础，重点加强工控系统安全防护和营销平台Web安全，构建"智慧烟草"安全底座。
