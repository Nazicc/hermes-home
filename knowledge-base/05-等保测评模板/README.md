## 使用方式

1. **新建项目：** 从 `1-资产梳理/` 获取资产调研表 → `7-定级备案/` 准备定级备案
2. **差距评估：** 从 `3-差距评估/差距评估表/` 选取对应等级的表单
3. **测评实施：** 参考 `3-差距评估/测评指导书/` 的操作指引
4. **整改：** 从 `4-整改方案/` 获取整改模板
5. **渗透测试：** 参考 `3-差距评估/渗透测试/` 样例报告格式

> ⚠️ 注意：所有文件为样例模板，使用时需替换为实际单位信息和系统信息
