# DLP数据防泄漏系统（TopDLP）

## 产品概述

天融信DLP数据防泄漏系统（TopDLP）是面向企业**数据安全防护**的专业产品，通过对数据**传输、存储、使用**全生命周期的监控与管控，防止敏感数据通过邮件、Web、即时通讯、存储介质、打印等渠道泄露。

TopDLP采用"**网络DLP + 终端DLP + 存储DLP + 邮件DLP**"四位一体的防护架构，内置深度内容识别引擎（指纹识别、关键词、正则表达式、自然语言处理、图像OCR等），支持对结构化数据和非结构化数据的精准识别与策略管控。满足数据安全法、个人信息保护法、等保2.0等合规要求。

## 核心功能

| 功能模块 | 说明 |
|---------|------|
| 网络DLP | 监控HTTP/HTTPS、SMTP、FTP、即时通讯等网络协议数据，实时阻断违规数据外传 |
| 终端DLP | 终端行为监控（打印、USB拷贝、截屏、剪贴板、蓝牙等），支持终端离线策略缓存 |
| 邮件DLP | 对WebMail和Outlook/Foxmail等客户端邮件进行内容审计与管控 |
| 存储DLP | 扫描文件服务器、NAS、SharePoint、云盘等存储位置的敏感数据分布 |
| 内容识别引擎 | 支持关键词、正则、文件指纹（精确/模糊）、数据指纹（结构化）、图像OCR、NLP分类 |
| 策略管理 | 基于用户/用户组、设备、网络位置、时间、内容等多维度的策略配置 |
| 事件审计 | 敏感数据流通行为全量审计，支持数据流向分析、用户行为画像 |
| 告警与阻断 | 支持审计模式（仅记录）和管控模式（阻断/加密/隔离），分级告警通知 |
| 脱敏展示 | 对审计结果中的敏感数据自动脱敏（如身份证号、手机号显示部分位） |

## 技术规格

| 参数项 | 规格 |
|-------|------|
| 网络吞吐 | 1Gbps ~ 20Gbps（取决于硬件规格） |
| 并发会话 | 50,000 ~ 500,000 |
| 内容识别规则 | 支持10,000+规则并发匹配 |
| 文件指纹库 | 支持百万级文件指纹 |
| 终端纳管 | 1,000 ~ 100,000+ 终端节点 |
| 协议支持 | HTTP/HTTPS、SMTP/SMTPS、FTP/FTPS、SMB、IMAP、POP3、即时通讯等30+协议 |
| 邮件类型 | Outlook、Foxmail、WebMail（QQ邮箱、网易、Gmail等）、Exchange |
| 数据指纹 | 支持精确指纹、模糊指纹、结构化指纹（数据库表/列） |
| 部署模式 | 网络DLP（网关/旁路）+ 终端DLP（Agent）+ 管理平台（集中管理） |

## 适用场景

- **金融行业**：客户身份信息、交易数据、信用卡信息防泄漏
- **政府机构**：涉密信息、内部公文、个人信息防外泄
- **运营商**：用户通信数据、计费数据、位置信息防泄露
- **医疗行业**：患者病历（PHI）、医疗影像数据防泄漏
- **教育和科研**：科研成果、学科数据、知识产权保护
- **涉密单位**：结合终端DLP实现涉密信息不上网、关键文件不外传

## 实战Tips（安服工程师经验）

**云贵区域特色：**

1. **贵州大数据企业敏感数据发现**：贵州大数据企业多，数据类型复杂（金融交易数据、政府共享数据、电商数据等）。建议先用**存储DLP扫描**功能对文件服务器和数据库进行一次全面扫描，形成敏感数据"家底清单"（数据分类分级），再针对高风险数据制定管控策略。

2. **云南旅游企业数据保护**：云南文旅企业掌握大量游客个人信息（身份证、护照、手机号、酒店住宿记录、航班信息等）。建议重点配置**个人信息保护策略**，凡是涉及批量导出/下载/外发游客信息的行为，必须触发审批或阻断。

3. **政务数据共享交换场景**：贵州作为国家政务数据共享试点省份，政府部门间数据交换频繁。建议DLP部署在**政务数据共享交换平台**出口侧，对跨部门数据传输实施内容审计，防止内部敏感信息意外外泄。

4. **终端DLP离线策略**：云南部分地州网络条件较差（如迪庆、怒江），笔记本用户经常离线办公。务必开启**终端DLP离线策略缓存**功能，确保终端在断网状态下仍然执行本地DLP策略，待网络恢复后自动上报审计日志。

5. **水印溯源**：建议开启文档**屏幕水印+打印水印**功能，当敏感文件被拍照外泄时可通过水印中的用户ID、设备ID、时间信息快速溯源责任人。

6. **分阶段上线策略**：DLP项目推荐分三阶段上线：第一阶段"纯审计模式"（仅记录不阻断，摸清数据流）、第二阶段"重点防护"（对高敏感数据管控）、第三阶段"全面防护"（全量数据分级管控）。避免一上来就阻断引发业务投诉。

7. **贵州本土行业特色**：贵州磷化、茅台等传统制造企业正在数字化转型，建议关注PLM/ERP/MES系统的数据防泄漏，这些系统包含研发设计图纸、生产工艺参数等核心商业机密。
