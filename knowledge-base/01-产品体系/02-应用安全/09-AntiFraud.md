# 反欺诈系统

## 产品概述

天融信反欺诈系统是天融信面向金融、政务、电商等行业推出的在线业务反欺诈安全产品，专注于检测与阻断业务层面的欺诈行为（包括账户盗用、虚假注册、信贷欺诈、支付欺诈、刷单套利、恶意抢兑等）。产品基于用户行为分析（UEBA）、设备指纹、风险评分引擎与实时决策引擎四大核心技术，在毫秒级完成每一次业务请求的风险评估与决策。

与传统网络安全产品关注"攻击"不同，反欺诈系统关注"欺诈"——攻击者通常伪装成正常用户完成"业务层面的恶意行为"。系统通过终端设备标识、生物行为特征（鼠标/键盘/触屏/滑动模式）、多维关联分析等技术识别"人"的身份真实性，而非仅仅验证"账号密码"的合法性，是金融机构和互联网平台业务风控体系的关键基础设施。

## 核心功能

| 功能模块 | 说明 |
|---------|------|
| **设备指纹** | 基于浏览器/设备硬件/网络信息的唯一设备标识，检测设备模拟器/改机工具/虚拟机/代理/VPN |
| **用户行为分析（UEBA）** | 鼠标轨迹、键盘输入节奏、触屏滑动模式、浏览路径等生物行为特征建模，识别非人操作 |
| **实时风险评分** | 基于规则引擎 + 机器学习模型，对每次业务请求进行 0-100 的风险评分，支持自定义阈值 |
| **场景化反欺诈** | 账号注册/登录/交易/转账/提现/绑卡/修改密码等场景独立策略配置 |
| **黑产情报库** | 云端黑产情报联动（代理 IP 库、黑手机号库、设备黑名单、虚拟 SIM 卡库等） |
| **关联图谱分析** | 基于图数据库的多维关联分析（同一设备多账号、同一 IP 多账户、相同收货地址等），发现欺诈团伙 |
| **核查与人工审核** | 高风险请求自动转人工审核，支持审核工单流转与协作 |
| **反欺诈报表** | 按日/周/月自动生成反欺诈报告，含欺诈事件趋势、场景分布、黑白名单更新统计 |

## 适用场景

- **金融交易反欺诈**：银行/证券/保险的在线交易欺诈检测，防止非授权资金转移与盗刷。
- **政务平台反欺诈**：社保/公积金/税务等政务服务的身份冒用与虚假申报检测。
- **电商反刷单**：电商平台促销活动的刷单/刷券/薅羊毛行为检测与阻断。
- **在线教育反作弊**：在线考试/测评的作弊行为检测（替考/作弊软件/AI 答题）。
- **P2P/网贷反欺诈**：借贷申请的信息伪造、团伙欺诈与多头借贷风险识别。

## 实战Tips（安服工程师经验）

> **云南区域经验：**
> - 云南普洱茶/鲜花等大宗电商交易平台的"薅羊毛"风控场景：每年双 11、618 大促期间，需临时开启"收货地址关联检测"——短时间内同一地址关联多个新注册账号下单的命中欺诈概率极高。建议规则：同一收货地址关联 3 个以上账号自动转人工审核。
> - 云南社保/医保线上服务平台（云南省医保公共服务平台）的反欺诈场景，重点关注"代理 IP + 新注册账号 + 大额报销申报"组合。建议反欺诈系统对医保报销类请求启用额外风控规则：新注册 7 天内不允许提交报销申请。
> - 云南景区（丽江/大理/西双版纳等热门景区）在线票务系统防黄牛抢票：建议启用"设备指纹 + 行为分析 + 历史购票记录"三重检测——同一个设备指纹在 24 小时内购买同一景区的门票超过 5 张自动触发人工审核。

> **贵州区域经验：**
> - 贵州茅台酒预约/申购系统的反黄牛场景是全国标杆级反欺诈需求。建议部署天融信反欺诈系统时配置多维度规则：①设备指纹检测（虚拟机/改机工具检测）；②收货地址黑名单（同一地址多人代购）；③关联图谱分析（同一设备->多账号->多身份证）。该配置大幅提升黄牛团伙识别率。
> - 贵州大数据交易所的用户认证与交易反欺诈：建议启用"行为基线"模式——系统学习用户 30 天的正常操作行为（登录时间、浏览页面、交易金额区间、操作间隔），当出现显著偏离基线时（如深夜大量交易、交易金额突增 10 倍）触发二次认证或人工审核。
> - 贵州政务服务平台（贵州政务服务网/多彩宝）的金融类业务反欺诈：建议反欺诈系统与公安人口库/实名认证系统联动，将初级认证用户（仅手机号注册）的业务权限限制为"查询类"，高级认证用户（人脸识别 + 身份证验证）才可办理资金类业务。

> **通用经验：**
> - 反欺诈系统上线初期最常见的错误是"误杀率过高"——过于激进的风控策略会大量阻断正常用户的业务操作，导致用户投诉和业务流失。建议采用"三阶段上线法"：第一阶段（1-2 周）仅告警不阻断，建立基线；第二阶段（1-2 周）阻断评分 ≥ 90 的高风险请求；第三阶段逐步收紧至目标阈值。
> - 设备指纹是最基础的反欺诈能力，但需要定期更新对抗黑产改机工具升级。建议设备指纹 SDK 每 3 个月进行防逆向加固更新，并与天融信云端指纹库保持同步。
> - 关联图谱分析对反欺诈的价值极大但计算成本也最高。建议将关联图谱的计算从"实时"降级为"近实时"（5-10 分钟延迟），确保实时决策的 P50 响应时间 ≤ 50ms。图谱关系日常离线计算，增量更新根据访问执行。
