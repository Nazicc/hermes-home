# SOAR安全编排自动化与响应

## 产品概述

天融信SOAR（Security Orchestration, Automation and Response）平台是面向安全运营**流程自动化**的下一代安全产品。通过将安全人员经验固化为**Playbook（剧本）**，以可视化拖拽方式编排安全处置流程，实现告警自动研判、威胁快速阻断、事件自动处置，大幅缩短MTTR（平均响应时间）。

SOAR定位于"安全运营的最后一公里"，承接SIEM/SOC产生的告警，联动各类安全设备（防火墙、EDR、沙箱、邮件网关等），通过"人+机器+流程"三位一体提升运营效率。典型效果：MTTR从小时级降至分钟级。

## 核心功能

| 功能模块 | 说明 |
|---------|------|
| 可视化剧本编排 | 拖拽式Playbook编辑器，支持条件分支、循环、并行、人工审批节点 |
| 原子动作库 | 内置300+预置操作（查杀、封堵IP、隔离主机、提取IOC、查询威胁情报等） |
| 设备联动编排 | 标准化API适配器对接天融信全系列及第三方100+厂商安全设备 |
| 告警自动化处置 | 告警触发自动执行剧本，支持黑白名单过滤、相似告警折叠 |
| 事件调查辅助 | 自动收集关联日志、提取上下文、生成调查报告草稿 |
| 工单自动化 | 自动生成ITIL工单，推送至工单系统，处置完成后自动闭环 |
| 案件管理 | 支持案件归类、标签、关联证据、溯源分析，形成完整案件时间线 |
| 度量与报表 | SOC运营KPI仪表盘（MTTD、MTTR、告警处置率、人工干预率等） |
| 多人协同 | 角色权限、操作留痕、评论协作、知识沉淀 |

## 适用场景

- **告警疲劳治理**：每天接收数千条告警的安全团队，自动过滤误报和低危告警
- **常见威胁自动处置**：暴力破解自动封堵源IP、勒索病毒自动隔离主机+提取IOC
- **钓鱼邮件响应**：邮件网关告警自动触发邮箱账号复核、用户确认、恶意链接分析
- **护网/攻防演练**：自动化剧本缩短应急处置窗口，配合人工审批关键操作
- **安全合规巡检**：定时自动执行资产核查、策略合规检查、结果输出报表

## 实战Tips（安服工程师经验）

**云贵区域特色：**

1. **贵州政务云SOAR联动**：贵州省市两级政务云环境下，安全设备品牌多（天融信、深信服、华为、奇安信等）。SOAR的**多厂商适配器**是刚需，建议优先测试与主流云安全组（Security Group）和WAF的API联动，确保封堵动作可达。

2. **云南边境安全应急**：瑞丽、磨憨等口岸城市面临跨境网络安全威胁。建议针对跨境攻击IP设计Playbook：识别跨境IP攻击 -> 自动查询威胁情报 -> 防火墙策略临时封堵 -> 通知当班人员确认 -> 24小时后自动解封。

3. **贵安数据中心集群SOAR**：在贵安超大规模数据中心，建议采用**SOAR集群部署**，并设置"熔断机制"——当单个剧本执行超过阈值（如100台主机同时隔离），自动触发人工审批，防止误操作导致大面积业务中断。

4. **护网值守场景**：云南护网期间，建议设置**分级处置策略**：低危告警全自动处置（不通知人）、中危告警自动处置+通知确认、高危告警自动收集情报+人工审批再执行。

5. **本地化IP黑名单**：建议建立"云贵专属IP信誉库"，通过提取本地护网/攻防演练中确认的恶意IP，定期导入SOAR的本地IOC库，提高剧本匹配精度。

6. **剧本灰度发布**：新剧本上线前先在测试环境运行，确认无异常后再推送到生产环境，避免因剧本逻辑缺陷影响现网业务。

7. **SOAR与SOC深度集成**：天融信SOAR与TopSOC/态势感知平台天然集成，建议采用"SOC告警触发->SOAR自动响应"的联动模式，效果最佳。
