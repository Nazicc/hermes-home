# 态势感知平台（TopSA）

## 产品概述

TopSA（TopSec Situational Awareness）是天融信推出的**新一代网络安全态势感知与预警平台**。产品基于大数据分析和可视化技术，从**全网视角**对资产、漏洞、威胁、事件、行为等多维信息进行实时汇聚、关联分析和智能研判，形成"可知、可见、可控、可管"的全局安全态势视图。

TopSA区别于传统SOC的核心在于：**强调预测性分析而非事后追溯**，通过机器学习建模对攻击趋势、风险变化进行预测预警，支持从宏观（省级/行业）到微观（单资产）的多层级态势下钻。

## 核心功能

| 功能模块 | 说明 |
|---------|------|
| 全网资产态势 | 实时呈现在线设备、存活主机、开放服务、资产风险分布热力图 |
| 威胁态势感知 | 攻击源TOP N、攻击类型分布、攻击路径溯源、威胁情报碰撞 |
| 漏洞态势 | 漏洞分布（按严重性/类型/厂商）、漏洞生命周期、修复进度跟踪 |
| 行为态势 | 用户异常行为分析（UEBA）、横向移动检测、特权账号异常监控 |
| 攻击链还原 | 结合MITRE ATT&CK框架，还原攻击杀伤链（侦察->武器化->交付->利用->安装->C2->目标） |
| 智能预警 | 基于时间序列预测模型，对安全态势趋势进行短中期预测 |
| 通报预警 | 支持向下级单位下发预警通报、整改通知书，形成闭环管理 |
| 行业监管对接 | 标准接口对接省级/国家级网络安全监管平台（如网信办、公安厅通报中心） |

## 技术规格

| 参数项 | 规格 |
|-------|------|
| 数据处理能力 | 50,000 ~ 200,000 EPS（取决于集群规模） |
| 数据存储 | 分布式存储（HDFS/ClickHouse），支持冷热分层，单集群PB级 |
| 分析引擎 | 流处理（Flink）+ 批处理（Spark），支持实时+历史混合分析 |
| 威胁情报 | 内置天融信TI情报源，支持STIX/TAXII标准接入第三方情报 |
| ML模型 | 内置异常检测、时间序列预测、聚类分析等10+算法模型 |
| 可视化 | HTML5零客户端大屏，支持多点触控、语音交互、地图热力 |
| API接口 | Restful API + WebSocket推送到第三方平台 |
| 部署方式 | 私有化部署 / 政务云部署 / 混合部署 |

## 适用场景

- **省级/地市级网络安全态势感知平台**：汇聚辖区内关键信息基础设施单位安全数据，统一态势呈现
- **行业监管态势**：金融监管、运营商、教育、卫健等行业态势感知与通报预警
- **重保期间指挥调度**：护网/攻防演练期间攻击态势实时监控与应急指挥
- **集团型企业安全运营**：多分支/多子公司统一安全态势感知与研判
- **安全托管服务（MSS）**：安全运营中心对外提供托管安全服务的统一展示平台

## 实战Tips（安服工程师经验）

**云贵区域特色：**

1. **贵州"中国数谷"数据汇聚**：贵阳大数据交易所和贵阳数博会相关单位需要建设跨行业态势感知平台。TopSA可通过标准数据交换接口（支持CSV、JSON、Syslog等格式）实现跨行业数据接入，建议采用**分级部署**模式：各单位本地部署采集探针，市级平台汇总分析，省级平台综合呈现。

2. **云南"一带一路"安全专区**：云南作为面向南亚东南亚辐射中心，跨境数据流动安全和边境网络安全需求突出。TopSA可针对**跨境流量**设置单独的攻击溯源分析策略，特别关注来自境外IP的攻击行为跟踪。

3. **贵州大数据安全特区**：贵安新区大数据安全产业示范区内的企业，建议TopSA对接**贵州大数据安全监管平台**，实现政企联动预警通报。

4. **云南文旅行业态势感知**：云南文旅产业发达（丽江、大理、西双版纳），旅游旺季景区票务系统、直播平台成为攻击热点。TopSA可针对订票系统API、支付网关配置专项态势面板。

5. **电力行业态势**：云南水电资源丰富（华能澜沧江、华电等），电力监控系统（电力调度、SCADA）态势感知需求强烈。TopSA支持工控协议（Modbus、IEC 104、OPC DA）的流量解析与异常检测。

6. **多源情报融合**：建议同时接入至少2-3个外部威胁情报源（如天际友盟、微步在线、奇安信威胁情报），与天融信TI情报交叉验证，减少误报。
