# 上网行为管理系统

> **产品定位**：天融信上网行为管理系统（TopACM / Web Behavior Management）是一款面向组织内部网络的互联网访问管控与审计产品。通过用户认证、应用控制、内容过滤和带宽管理，帮助企业实现**合规上网**、**工作效率提升**和**上网安全防护**三重目标。

---

## 一、产品概述

天融信上网行为管理系统基于深度包检测（DPI）技术，能够识别 3,000+ 应用和 5,000 万+ URL 分类，支持 AD/LDAP/RADIUS/Portal 等多种用户认证方式，将流量与用户身份关联，实现精细化的上网权限控制。产品满足《网络安全法》、《数据安全法》以及等级保护 2.0 对审计和日志留存的法律要求。

产品分为硬件一体机和虚拟化两种交付形态，覆盖 200 人以下的小型单位到 10 万人以上的大型组织。

## 二、核心功能

| 功能分类 | 功能说明 |
|---------|---------|
| **用户认证与准入** | 支持 AD/LDAP/RADIUS/本地数据库/Portal/短信认证/SAML SSO，支持终端 MAC 绑定和终端合规性检查 |
| **应用识别与控制** | 识别 3,000+ 应用（含移动端 App），按用户/用户组/时间段粒度做允许/拒绝/限速策略 |
| **URL 分类过滤** | 内置 5,000 万+ URL 分类库，覆盖 85+ 分类（色情、赌博、暴力、社媒、影音等），支持自定义 URL 黑白名单 |
| **内容审计** | 审计邮件（SMTP/POP3/Webmail）、HTTP/HTTPS POST 内容、论坛发帖、即时通信（微信/企微/钉钉/飞书）消息日志 |
| **带宽管理** | 每用户/每应用/每时间段的 QoS 流量控制，支持通道化配置（保证关键业务带宽） |
| **行为分析报表** | 提供 100+ 预置报表（用户排行、应用排行、热点事件、趋势分析），支持自定义报表 |
| **非法热点检测** | 检测私接路由器/随身 WiFi 等非法热点行为，防止网络边界失控 |
| **SSL 解密审计** | 支持对 HTTPS 加密流量做中间人解密审计（需客户端安装根证书） |
| **日志留存** | 满足《网络安全法》180 天日志留存要求，支持本地/外置/云存储归档 |

## 四、适用场景

### 4.1 政府机关办公上网
管控公务人员的非工作上网行为（如刷抖音、看视频、炒股），防止泄密和违规外联。

### 4.2 企业/园区网出口
对员工上网行为做审计和带宽管理，保障关键业务（OA、ERP、视频会议）带宽质量。

### 4.3 学校/教育城域网
管控学生上网权限（屏蔽不良信息、限制游戏/视频），满足教育部校园网络安全要求。

### 4.4 酒店/公共 WiFi
满足《互联网安全保护技术措施规定》（公安部 82 号令）对公共场所上网实名认证和日志留存的要求。

## 五、实战 Tips（安服工程师经验）

### 🎯 云贵区域特色

1. **云南中小学及高校**：云南多地教育局采购上网行为管理用于校园网出口。建议：
   - 开启 **色情/暴力 URL 过滤**（必选，满足教育部校园安全要求）
   - 开启 **未成年人保护模式**（晚 22:00-早 06:00 断网）
   - 特别：云南少数民族地区学校建议本地化添加少量本地语不良关键词的黑名单

2. **贵州政务办公场景**：贵州省政府单位推行"互联网+政务服务"，办公网络需要同时满足"外网可访问"和"行为有审计"的矛盾需求。建议采用 **白名单模式**——只允许与工作相关的 URL（如政务服务平台、OA、邮箱），其余一律阻断。配合 SSL 解密可审计 HTTPS 政务流量。

3. **云贵旅游酒店行业**：丽江、大理、西双版纳的酒店和民宿面临公安网安检查。建议：
   - 开启 **实名认证**（微信认证或短信认证，配合公安部 82 号令）
   - 留存日志 ≥ 180 天，存储空间建议按日均 2GB 规划（300 间客房规模）
   - 注意：旅游旺季流量暴增（可能达到平时 5-10 倍），需确认设备吞吐余量

4. **通用建议**：
   - **认证对接**：如果客户已有 AD 域控，强烈推荐对接 AD 认证，可以实现"人"维度的行为审计（访客走 Portal 认证即可）。
   - **带宽策略**：建议为视频会议（腾讯会议/Zoom/钉钉会议）设置 **保证带宽**（≥ 50% 出口带宽），其余应用共享剩余带宽。
   - **隐私问题**：内容审计功能请务必获得法务部门和工会的书面同意，避免法律风险。建议只审计 HTTP POST 内容的元数据（URL 和收发双方），不对邮件正文做全文存储。
   - **云贵跨境流量**：靠近边境地区的客户出口流量可能涉及跨境 VPN 隧道，注意上网行为管理不要阻断合法的跨境专线流量。
