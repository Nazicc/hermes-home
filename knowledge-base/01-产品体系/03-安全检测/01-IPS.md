# TopIPS 入侵防御系统

> **产品定位**：天融信网络入侵防御系统（TopIPS）是一款面向网络层和应用层的实时深度检测与防御产品，部署于网络关键节点，对传输流量进行逐包分析和实时阻断，是纵深防御体系中的"主动防御核心"。

---

## 一、产品概述

TopIPS 基于天融信自主知识产权的 TOP 系列硬件平台和 TOS（Topsec Operating System）操作系统，采用**深度包检测（DPI）**与**深度流检测（DFI）**双擎技术架构，融合**入侵检测（IDS）**与**入侵防御（IPS）**双重能力于一体。产品可识别超过 6,500+ 已知攻击特征，覆盖漏洞利用、病毒传播、木马回连、SQL 注入、XSS 攻击、命令执行、文件包含等主流攻击类型。

TopIPS 是等级保护 2.0 三级以上系统中**必选的安全设备**，广泛应用于政府、金融、教育、医疗、运营商、能源等行业。

## 二、核心功能

| 功能分类 | 功能说明 |
|---------|---------|
| **攻击检测与阻断** | 基于特征库 + 异常行为分析，实时检测并拦截漏洞利用、扫描探测、暴力破解、Web攻击等 |
| **应用识别与控制** | 识别 3,000+ 应用协议（含加密流量），基于用户/应用/时间做精细化访问控制 |
| **病毒查杀** | 集成反病毒引擎，对流经文件进行实时病毒扫描与阻断 |
| **热点事件规则** | 针对高危漏洞（如 Log4j、Struts2、WebLogic）下发紧急虚拟补丁，无需等待厂商正式补丁 |
| **SSL/TLS 解密** | 支持对加密流量（HTTPS/SSL/TLS）的中间人解密检测，覆盖"加密躲猫猫"攻击 |
| **威胁情报联动** | 支持对接天融信云端威胁情报中心（TI Center），实时拉取失陷指标（IOC）并自动生成阻断规则 |
| **DoS/DDoS 防护** | 支持 SYN Flood、UDP Flood、ICMP Flood、DNS Query Flood 等常见 DDoS 攻击的检测与清洗 |
| **流量可视化** | 实时展示 Top N 攻击源/目的、应用分布、带宽占用、会话状态，支持 N×24 小时历史回溯 |

## 四、适用场景

### 4.1 互联网出口防护
在企业/机构互联网出口部署，作为"第一道防线"防御来自外网的攻击和恶意扫描。

### 4.2 数据中心边界防护
部署于数据中心核心交换机前，对南北向流量进行深度检测，阻断针对业务系统的漏洞利用攻击。

### 4.3 分支机构互联防护
在总部与分支的广域网链路中串联部署，防止横向移动攻击在分支间扩散。

### 4.4 等级保护合规
满足等保 2.0 三级及以上系统的入侵防范要求（GB/T 22239-2019 安全计算环境/安全区域边界）。

## 五、实战 Tips（安服工程师经验）

### 🎯 云贵区域特色

1. **云南教育行业（高校）**：云贵高校普遍存在 IP 地址不足、IPv6 过渡阶段问题。建议 IPS 开启 **IPv6 安全策略**，并配置 **DDoS 防护**（教育网出口经常遭受扫描攻击）。特别推荐开启 **SQL 注入 + 跨站脚本** 规则集——高校招生系统、教务系统往往存在大量 Web 漏洞。

2. **贵州政务云场景**：贵州作为国家大数据综合试验区，政务云平台密集。IPS 部署建议采用 **虚拟化版本（vIPS）** 接入 VPC 内部，配合 **SSL 解密** 检测政务 HTTPS 流量中的数据窃取行为。注意：贵州政务服务网流量大，推荐选型不低于 10Gbps 吞吐的硬件型号。

3. **云南烟草行业**：烟草行业信息化系统多为内网应用，IPS 需要重点开启 **ICS 工业协议检测** 规则集（MODBUS、S7Comm、IEC 60870-5-104），防止针对 SCADA 控制系统的恶意攻击。

4. **通用建议**：
   - 上线初期建议设置为 **告警模式（Alert）** 运行 1-2 周，确认无误报后再切换到 **阻断模式（Prevent）**。
   - 特征库务必开启 **每日自动更新**，高危漏洞爆发时手动触发紧急更新。
   - 定期（建议每月）导出 IPS 日志做攻击趋势分析，形成区域安全态势感知的底层数据。
   - 注意：云贵高海拔地区（如香格里拉、丽江），部分 IPS 硬件散热需确认机柜空调是否充足，防止高温宕机。
